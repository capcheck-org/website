# Contributing to CapCheck

Thank you for your interest in contributing to CapCheck!

## Contributor License Agreement (CLA)

Before we can accept your contributions, you must sign our Contributor License Agreement.

**Please complete the CLA form here: [https://capcheck.org/cla](https://capcheck.org/cla)**

The CLA ensures that:
- You have the right to submit the contribution
- You grant us a license to use your contribution
- The project can be properly licensed and distributed

## How to Contribute

1. **Sign the CLA** at [https://capcheck.org/cla](https://capcheck.org/cla)
2. Fork the repository
3. Create a feature branch (`git checkout -b feature/your-feature`)
4. Make your changes
5. Commit your changes with clear, descriptive messages
6. Push to your fork
7. Open a Pull Request

## Questions?

If you have questions about contributing, please open an issue in the repository.
